# Pilot Consent (Template)

**Project:** Smart Pharmacy Inventory System (SPIS) — Pilot Program

**Purpose:**  
This pilot will test automated inventory forecasting and restock recommendations for pharmacies.  
We will collect anonymized data from your store to improve forecast accuracy.

**Data Usage:**  
- Only **inventory and sales data** will be used.  
- No personally identifiable information (PII) about customers or staff will be shared.  
- Data will be stored securely and accessed only by authorized project staff.

**Data Protection:**  
- All data will be stored using industry-standard encryption.
- Backups are encrypted and stored on secure cloud infrastructure.
- Data will be deleted within **90 days after the pilot ends** unless renewed consent is obtained.

**Duration:**  
The pilot will run for approximately **6 months**.
